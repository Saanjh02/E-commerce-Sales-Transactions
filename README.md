# 📊 E-Commerce Sales Transactions 

## 📌 Project Overview
This project is an end-to-end **E-Commerce Sales Transactions** built using **SQL Server and Power BI**. It analyzes sales performance, customer behavior, product categories, and regional trends to generate actionable business insights.

---

## 🛠️ Tools & Technologies
- SQL Server
- SQL (Data Cleaning & Analysis)
- Power BI
- DAX (Data Analysis Expressions)
- Data Visualization

---

## 📂 Files in this Repository
- `dashboard.pbix` → Power BI dashboard file
- `dataset.csv` → Raw dataset used for analysis
- `sql_queries.sql` → SQL queries for data cleaning and analysis

---

## 🔄 Project Workflow

### 1. Data Cleaning (SQL)
- Removed duplicates
- Handled missing values
- Verified data consistency

### 2. Data Analysis (SQL)
- Used JOINs, GROUP BY, aggregations
- Extracted business insights from raw data

### 3. Data Modeling (Power BI)
- Imported cleaned data into Power BI
- Created relationships between tables

### 4. DAX Measures
- Total Revenue
- Total Orders
- Total Customers
- Average Order Value (AOV)

### 5. Dashboard Development
Built interactive visuals including:
- KPI Cards
- Revenue by Category
- Revenue by Region
- Monthly Revenue Trend
- Top Customers Analysis

---

## 📊 Key Insights
- Electronics category generated the highest revenue
- Certain regions contributed significantly more to sales
- Revenue showed consistent monthly performance
- A small group of customers contributed major revenue share
